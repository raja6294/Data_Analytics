## 📂 Projects Included

### 🔹 1. Netflix Data Analysis

* Analyzed Netflix dataset to find:

  * Content distribution (Movies vs TV Shows)
  * Year-wise growth
  * Top genres
  * Country-wise content trends
* Used visualizations like bar charts, pie charts, and histograms

---

### 🔹 2. IPL Data Analysis

* Performed analysis on Indian Premier League dataset
* Key insights:

  * Team performance and win ratios
  * Toss impact on match results
  * Top players (batsmen & bowlers)
  * Match trends and patterns
